## Algorithm Lecture Summary - Section1 구간 합
### ✏️ Study
#### 💡 구간 합

> **구간 합이란 ?**
> 
- 합 배열을 이용하여 시간 복잡도를 더 줄이기 위해 사용하는 특수한 목적의 알고리즘

> **합 배열**
> 
- 배열 A가 있을 때 합 배열 S의 정의

$S[i] = A[0] + A[1] + ... + A[i-1] + A[i]$

- 합 배열 S를 만드는 공식

$S[i] = S[i-1] + A[i]$

- 구간 합을 구하는 공식

$S[j] - S[i-1]$      // i에서 j까지의 구간 합
